About Rocky

Rocky is an interpreter of the Rockstar programming language, fully compliant with the spec, written in Java. Rocky offers many unique features like:

  • Object-Oriented Programming
  • Native JAVA binding (available only offline)
  • Support for different math modes: IEEE754 (double precision), BigDecimal (infinite precision) or Dec64
  • CLI Debugging
  • Extended features like aliasing, advanced array handling
