Where to find Rocky?

Rocky is available on Github via https://github.com/gaborsch/rocky.

How to install it offline?

The easiest way is if you clone the repository: git clone https://github.com/gaborsch/rocky.git. After installing locally, you can enjoy the CLI debugger, play with different options, and you may even use the native Java binding to run Java code from Rocky!

To execute your Rockstar poem, you can run rockstar.bat my-poem.rock (Windows) or ./rockstar my-poem.rock (Linux). You must have at least Java8 JRE installed.

On Linux, you can execute install.sh to install /usr/bin/rockstar as a standalone executable.

I'm stuck with my code!

There is a help for every command. Try rockstar help run for example.

If the syntax is messy, try rockstar list my-poem.rock to list, or rockstar list -x my-poem.rock to get your code explained.

The language specification is available here.

How to use the debugger?

Start your program with rockstar debug my-poem.rock. It will execute your program step-by-step, printing each statement before executing. Enter a debug command and press Enter to continue.
At the command prompt enter ? to get more help.

  • Step Into / Step Over / Step Return or Run with 5, 6, 7, 8 (Pressing Enter without command means Step Into)
  • Explain expression evaluation with x or 1
  • Show all variable with s
  • Print the statement to execute with . (no execution)
  • List your whole program with list or list -x
  • Exit to command prompt with exit
